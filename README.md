# das web poc


# usage

- `dasContDev.exe N_<name>.das`
- `node websockify.js localhost:1000 localhost:9000`
- open in browser `N_<name>.html`

- *.vue component from sample #7 requires local http server, for example `python -m SimpleHTTPServer`
